# Python Based Mini Projects

This repository contains a collection of beginner-friendly Python mini projects created to practice core programming concepts such as logic building, loops, conditionals, file handling, and object-oriented programming (OOPs).

## Projects Included

### 1. Number Guessing Game
- A simple console-based game
- Uses random number generation and loops
- Helps in understanding conditional logic and user input handling

### 2. File Handling Mini Project
- Demonstrates basic file operations in Python
- Includes create, read, write, update, and delete operations
- Implemented using a menu-driven approach

### 3. Bank Management System
- A Python-based banking application
- Built using OOP concepts and file handling
- Supports basic operations like account creation, deposit, withdrawal, and balance inquiry

## Technologies Used
- Intermediate Python
- Python OOPs
- File Handling

This repository is created for learning purposes and to strengthen Python fundamentals through hands-on projects.
