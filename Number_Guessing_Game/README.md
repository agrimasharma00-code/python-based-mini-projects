# Number Guessing Game (Python)

This is a simple **Number Guessing Game** developed using **Python**.  
The project was created to practice basic Python concepts such as loops, conditionals, and user input handling.

## Project Description

In this game:
- The system generates a random number.
- The user tries to guess the number.
- The program gives hints like **Too High** or **Too Low**.
- The game continues until the correct number is guessed.
This project is beginner-friendly and helps in understanding logical thinking using Python.
